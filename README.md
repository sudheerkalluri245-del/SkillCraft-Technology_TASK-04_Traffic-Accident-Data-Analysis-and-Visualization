# SkillCraft Technology – Data Science Internship

## Task 04: Traffic Accident Data Analysis and Visualization


##  Project Overview

As part of the **SkillCraft Technology Data Science Internship**, I successfully completed **Task 04: Traffic Accident Data Analysis and Visualization**. The objective of this project was to analyze traffic accident data to identify patterns related to road conditions, weather, and time of day while visualizing accident hotspots and contributing factors.

The entire project was developed and executed using **Google Colab**, where Python libraries were used to clean, analyze, and visualize accident data. Through this project, I gained practical experience in Exploratory Data Analysis (EDA), data visualization, pattern recognition, and extracting meaningful insights from real-world datasets.



##  Objective

The primary objectives of this project were:

- Import and understand the traffic accident dataset.
- Perform data cleaning and preprocessing.
- Analyze accident patterns based on road conditions.
- Study the impact of weather conditions on accidents.
- Analyze accidents by time of day.
- Identify accident hotspots.
- Create meaningful visualizations.
- Discover contributing factors affecting road accidents.
- Extract valuable insights from the dataset.
